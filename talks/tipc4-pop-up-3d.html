Why are we here?

  • Main objective: to digitize and create interactive visualizations of historical game books.
    • All the objects of the collection get digitized in video.
    • All possible objects get scanned.
    • Some objects get a 3D model.
    • All 3D models with kinetic properties get animated.
    • Display and user interaction get implemented in Kompakkt.
  • Long term: creating potential standards for kinetic 3D digitizations and visualization.
